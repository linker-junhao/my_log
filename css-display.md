# css属性之dispaly
display属性可以控制元素的内部和外部显示类型，主要分为内部显示类型（inner display type）和外部显示类型（outer display type）两种。

其关键字取指被分为6类：`display-outside`, `display-inside`, `display-list-item`, `display-internal`, `display-box`, `display-legacy`。

## display-outside
定义了元素的外部显示类型。

### block
以块级元素显示。

### inline
以内联元素显示。
